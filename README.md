# 石頭大師 Stone Master

一款結合 AR 實境探索、AI 石頭辨識、石頭角色養成、策略戰鬥與多人社交的手機遊戲。玩家從現實世界掃描石頭，AI 分析外型、顏色、紋理生成專屬 3D 角色，培養、裝扮、戰鬥，成為世界級石頭大師。

- 平台：iOS / Android
- 引擎：Unity + AR Foundation（規劃中，見技術架構文件）
