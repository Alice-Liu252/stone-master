# AI 石頭百科 — 種子資料（第二批，8 種）

對應 [../GDD.md](../GDD.md) 第 5 章「AI 石頭百科系統」的欄位規格，延續 [ENCYCLOPEDIA_SAMPLE.md](ENCYCLOPEDIA_SAMPLE.md) 已涵蓋的 10 種條目，新增 8 種尚未收錄的岩石與礦物，同樣涵蓋火成岩／沉積岩／礦物三大類，並優先選擇具台灣在地地質意義的種類。

## 11. 安山岩 Andesite

| 欄位 | 內容 |
|---|---|
| 中文名稱 | 安山岩 |
| 英文名稱 | Andesite |
| 學名 | Andesite |
| 分類 | 火成岩（噴出岩，中性岩） |
| 顏色 | 灰、深灰至黑色 |
| 紋理 | 細粒基質中崁有較大礦物顆粒的斑狀結構 |
| 硬度 | 約 6（整體，因含多種礦物） |
| 密度 | 2.5～2.8 g/cm³ |
| 光澤 | 暗淡至微光 |
| 成分 | 富含斜長石、角閃石、輝石，二氧化矽含量介於玄武岩與流紋岩之間 |
| 形成方式 | 中性岩漿噴出地表快速冷卻凝固，常見於隱沒帶火山 |
| 地質年代 | 台灣北部火山活動主要發生於更新世 |
| 世界產地 | 環太平洋火山帶（日本、印尼、南美安地斯山脈） |
| 台灣產地 | 大屯火山群（陽明山）、龜山島、蘭嶼、綠島 |
| 收藏價值 | 一般，具明顯斑狀結構的標本較具教育展示價值 |
| 用途 | 建材、火山地質教育 |
| 趣味知識 | 「安山岩」（Andesite）的英文名稱正是源自南美洲安地斯山脈（Andes），是全球隱沒帶火山最常見的岩石種類之一 |

## 12. 砂岩 Sandstone

| 欄位 | 內容 |
|---|---|
| 中文名稱 | 砂岩 |
| 英文名稱 | Sandstone |
| 學名 | Sandstone |
| 分類 | 沉積岩 |
| 顏色 | 黃褐、灰白、淺黃等，依膠結物與砂粒成分而異 |
| 紋理 | 砂粒膠結而成，可見層理構造 |
| 硬度 | 依膠結程度而異，約 3～6 |
| 密度 | 2.2～2.7 g/cm³ |
| 光澤 | 無特定光澤，表面粗糙 |
| 成分 | 石英、長石等砂粒，經矽質、鈣質或鐵質膠結物固結而成 |
| 形成方式 | 河流、海濱環境的砂粒經長期壓密與膠結作用固結而成 |
| 地質年代 | 台灣西部麓山帶砂岩多形成於中新世至上新世 |
| 世界產地 | 全球沉積盆地皆常見 |
| 台灣產地 | 東北角海岸（瑞芳深澳）、台灣西部丘陵地區廣泛分布 |
| 收藏價值 | 一般，具特殊風化紋理或化石痕跡者較特別 |
| 用途 | 建材、東北角海岸奇岩地景教育 |
| 趣味知識 | 新北市瑞芳深澳岬角著名的「豆腐岩」地景，是厚層砂岩沿兩組互相垂直的節理受海水侵蝕，切割成豆腐般方格狀的特殊地形 |

## 13. 泥岩 Mudstone

| 欄位 | 內容 |
|---|---|
| 中文名稱 | 泥岩 |
| 英文名稱 | Mudstone |
| 學名 | Mudstone |
| 分類 | 沉積岩 |
| 顏色 | 青灰色、灰色 |
| 紋理 | 顆粒極細緻密，缺乏明顯層理 |
| 硬度 | 低，約 2～3，遇水易軟化崩解 |
| 密度 | 2.0～2.4 g/cm³ |
| 光澤 | 無光澤，暗淡 |
| 成分 | 極細泥質顆粒（黏土礦物與粉砂），碳酸鈣含量較高 |
| 形成方式 | 古代海底深水環境中極細泥質顆粒緩慢沉積固結而成 |
| 地質年代 | 台灣西南部泥岩主要形成於上新世至更新世 |
| 世界產地 | 全球深海沉積盆地皆可見類似泥岩 |
| 台灣產地 | 高雄田寮、燕巢，台南龍崎一帶（俗稱「月世界」地形） |
| 收藏價值 | 低，質地鬆軟不易長期保存為標本 |
| 用途 | 地質教育、惡地地形觀光 |
| 趣味知識 | 泥岩質地緻密不透水又極易受雨水沖蝕，加上稀疏植被難以附著，造就了高雄田寮、燕巢一帶寸草不生的荒涼「月世界」惡地地形，是台灣獨特的地質奇觀 |

## 14. 珊瑚礁石灰岩（咾咕石） Coral Limestone

| 欄位 | 內容 |
|---|---|
| 中文名稱 | 珊瑚礁石灰岩（咾咕石） |
| 英文名稱 | Coral Reef Limestone |
| 學名 | Reef Limestone |
| 分類 | 沉積岩（生物化學沉積岩） |
| 顏色 | 灰白、米黃，常保留珊瑚孔洞結構 |
| 紋理 | 多孔隙，可見珊瑚骨骼與貝殼碎片痕跡 |
| 硬度 | 約 3（方解石為主要成分） |
| 密度 | 2.3～2.6 g/cm³（因多孔隙常較純方解石輕） |
| 光澤 | 無特定光澤，表面粗糙多孔 |
| 成分 | 主要為碳酸鈣 CaCO₃，由珊瑚骨骼與貝殼碎屑膠結而成 |
| 形成方式 | 古代珊瑚礁與貝殼經長期膠結、部分經地殼抬升出露地表而成 |
| 地質年代 | 台灣珊瑚礁石灰岩多為更新世至全新世 |
| 世界產地 | 全球熱帶珊瑚礁海岸皆可見（如加勒比海、東南亞島嶼） |
| 台灣產地 | 恆春半島、澎湖群島、小琉球、東沙環礁 |
| 收藏價值 | 一般，多孔隙標本具展示教育價值 |
| 用途 | 傳統建材（澎湖「咾咕石」厝的牆體建材）、珊瑚礁地質教育 |
| 趣味知識 | 澎湖傳統民居常見「咾咕石牆」，居民就地取材以珊瑚礁石灰岩堆砌牆面，是海島型建築的重要文化資產 |

## 15. 閃玉（豐田玉） Nephrite (Taiwan Jade)

| 欄位 | 內容 |
|---|---|
| 中文名稱 | 閃玉（豐田玉） |
| 英文名稱 | Nephrite (Taiwan Jade) |
| 學名 | Nephrite（鈣鎂鐵閃石系列） |
| 分類 | 礦物（閃石族） |
| 顏色 | 墨綠、蘋果綠、白色等，依鐵含量而異 |
| 紋理 | 纖維交織結構，質地細密堅韌 |
| 硬度 | 莫氏 6～6.5 |
| 密度 | 2.9～3.0 g/cm³ |
| 光澤 | 油脂光澤 |
| 成分 | 鈣鎂鐵閃石類礦物（透閃石—陽起石系列） |
| 形成方式 | 蛇紋岩與圍岩接觸交代作用生成 |
| 地質年代 | 與花蓮蛇紋岩地質帶同期 |
| 世界產地 | 中國新疆和闐、加拿大卑詩省、紐西蘭 |
| 台灣產地 | 花蓮豐田 |
| 收藏價值 | 高——1960～1970 年代台灣曾是全球主要閃玉輸出地之一，優質標本仍具收藏價值 |
| 用途 | 玉石雕刻、飾品、外銷加工 |
| 趣味知識 | 1960～70 年代台灣閃玉（豐田玉）產量曾居全球前列並大量外銷，是台灣礦業史上重要的一頁，後因蘊藏量減少、開採成本上升而逐漸式微 |

## 16. 方解石 Calcite

| 欄位 | 內容 |
|---|---|
| 中文名稱 | 方解石 |
| 英文名稱 | Calcite |
| 學名 | Calcite (CaCO₃) |
| 分類 | 礦物（碳酸鹽礦物） |
| 顏色 | 無色、白色，常因雜質呈黃、粉、灰等色 |
| 紋理 | 菱面體解理發達，晶體形態多樣 |
| 硬度 | 莫氏 3 |
| 密度 | 約 2.71 g/cm³ |
| 光澤 | 玻璃光澤 |
| 成分 | 碳酸鈣 CaCO₃ |
| 形成方式 | 石灰岩地區地下水溶蝕後再沉積，或熱液礦脈中結晶 |
| 地質年代 | 各地質年代皆可生成，鐘乳石洞內持續形成中 |
| 世界產地 | 全球石灰岩地區皆常見 |
| 台灣產地 | 恆春半島石灰岩洞穴、太魯閣大理岩區 |
| 收藏價值 | 中等，具完整菱面體晶形或特殊晶簇者價值較高 |
| 用途 | 水泥與建材原料、鐘乳石景觀教育、光學儀器（冰洲石） |
| 趣味知識 | 方解石具有「雙折射」特性，透明無瑕的冰洲石變種曾被用於製造偏光顯微鏡等光學儀器，是辨識方解石最有趣的物理特性之一 |

## 17. 黃鐵礦 Pyrite

| 欄位 | 內容 |
|---|---|
| 中文名稱 | 黃鐵礦 |
| 英文名稱 | Pyrite |
| 學名 | Pyrite (FeS₂) |
| 分類 | 礦物（硫化物礦物） |
| 顏色 | 淺黃銅色，具強金屬光澤 |
| 紋理 | 常見立方體或五角十二面體晶形 |
| 硬度 | 莫氏 6～6.5 |
| 密度 | 約 5.0 g/cm³ |
| 光澤 | 強金屬光澤 |
| 成分 | 二硫化亞鐵 FeS₂ |
| 形成方式 | 熱液礦脈、沉積岩與變質岩中皆可生成，常與金礦脈伴生 |
| 地質年代 | 各地質年代皆有生成 |
| 世界產地 | 西班牙、秘魯、美國等地大型硫化礦床 |
| 台灣產地 | 新北市金瓜石、九份地區金礦脈中常見伴生礦物 |
| 收藏價值 | 中等，具完整立方體晶形的標本收藏價值較高 |
| 用途 | 硫酸工業原料、早期作為採金指標礦物 |
| 趣味知識 | 因外觀酷似黃金而被稱為「愚人金」（Fool's Gold），金瓜石早期採礦人常需藉由比重與硬度判斷，避免誤將黃鐵礦當作真金 |

## 18. 硫磺 Sulfur

| 欄位 | 內容 |
|---|---|
| 中文名稱 | 硫磺 |
| 英文名稱 | Sulfur |
| 學名 | Native Sulfur (S) |
| 分類 | 礦物（自然元素礦物） |
| 顏色 | 鮮黃色 |
| 紋理 | 常呈細粒狀、鐘乳狀或結晶附著於岩石表面 |
| 硬度 | 莫氏 1.5～2.5 |
| 密度 | 約 2.0～2.1 g/cm³ |
| 光澤 | 樹脂光澤 |
| 成分 | 純硫元素 S |
| 形成方式 | 火山噴氣孔硫化氫氣體與氧接觸氧化結晶而成 |
| 地質年代 | 現代火山活動持續生成中 |
| 世界產地 | 環太平洋火山帶各火山地熱區 |
| 台灣產地 | 大屯火山群大磺嘴、小油坑、北投地熱谷周邊 |
| 收藏價值 | 低，質地脆弱易碎不易長期保存 |
| 用途 | 溫泉地質教育、早期火藥與化工原料開採 |
| 趣味知識 | 清代文獻已記載北投、大屯山一帶「硫磺穴」，郁永河《裨海紀遊》即描述來台採硫的經過，是台灣現存最早的採礦紀錄之一 |

## 分類統計（本批新增）

| 岩石/礦物種類 | 涵蓋條目 |
|---|---|
| 火成岩 | 安山岩 |
| 沉積岩 | 砂岩、泥岩、珊瑚礁石灰岩（咾咕石） |
| 礦物 | 閃玉（豐田玉）、方解石、黃鐵礦、硫磺 |

台灣在地產地新增涵蓋：陽明山／龜山島／蘭嶼（安山岩）、瑞芳深澳（砂岩）、高雄田寮燕巢（泥岩）、恆春／澎湖／小琉球（咾咕石）、花蓮豐田（閃玉）、恆春／太魯閣（方解石）、金瓜石九份（黃鐵礦）、北投大屯山（硫磺）——與 [ENCYCLOPEDIA_SAMPLE.md](ENCYCLOPEDIA_SAMPLE.md) 合計 18 種條目，可對應 GDD 第 11 章「探索與召喚系統」的地區分布設計。
