# AI 石頭百科 — 種子資料（10 種範例）

對應 [../GDD.md](../GDD.md) 第 5 章「AI 石頭百科系統」的欄位規格。這是知識庫的第一批範例條目，涵蓋岩石三大類（火成岩／沉積岩／變質岩）＋礦物，並特別放入一顆台灣特有的稀有礦物。

> **審核狀態**：草稿。內容依公開地質學常識整理，正式上線前仍需地質內容顧問逐條審核校對（見 GDD 5.3「內容可信度原則」），避免教育內容出錯。

## 1. 石英 Quartz

| 欄位 | 內容 |
|---|---|
| 中文名稱 | 石英 |
| 英文名稱 | Quartz |
| 學名 | Quartz (SiO₂) |
| 分類 | 礦物（氧化物礦物） |
| 顏色 | 無色、白、煙灰、粉紫等，依雜質而異 |
| 紋理 | 玻璃光澤，貝殼狀斷口 |
| 硬度 | 莫氏 7 |
| 密度 | 2.65 g/cm³ |
| 光澤 | 玻璃光澤 |
| 成分 | 二氧化矽 SiO₂ |
| 形成方式 | 火成岩結晶、熱液礦脈、變質作用皆可生成 |
| 地質年代 | 各地質年代皆有生成 |
| 世界產地 | 巴西、馬達加斯加、美國阿肯色州 |
| 台灣產地 | 花蓮、南投等地溪床常見 |
| 收藏價值 | 中等，透明無瑕者價值較高 |
| 用途 | 玻璃製造、電子工業（壓電石英）、飾品 |
| 趣味知識 | 沙子的主要成分就是石英顆粒；早期水晶收音機正是利用石英的壓電效應 |

## 2. 紫水晶 Amethyst

| 欄位 | 內容 |
|---|---|
| 中文名稱 | 紫水晶 |
| 英文名稱 | Amethyst |
| 學名 | Amethyst（石英變種，SiO₂ 含微量鐵） |
| 分類 | 礦物 |
| 顏色 | 紫色，濃淡因鐵離子含量與天然輻射而異 |
| 紋理 | 柱狀結晶，常見於晶洞內壁 |
| 硬度 | 莫氏 7 |
| 密度 | 2.65 g/cm³ |
| 光澤 | 玻璃光澤 |
| 成分 | SiO₂ 含微量 Fe |
| 形成方式 | 火山岩晶洞（geode）中低溫熱液緩慢結晶 |
| 地質年代 | 各地質年代皆有 |
| 世界產地 | 巴西、烏拉圭、尚比亞 |
| 台灣產地 | 天然大型礦床罕見，市面多為進口 |
| 收藏價值 | 高，色澤濃郁均勻者尤佳 |
| 用途 | 珠寶飾品、擺飾 |
| 趣味知識 | 曾被歐洲皇室視為與紅寶石同等珍貴的寶石，直到 19 世紀巴西發現大量礦床後價值才下降 |

## 3. 花崗岩 Granite

| 欄位 | 內容 |
|---|---|
| 中文名稱 | 花崗岩 |
| 英文名稱 | Granite |
| 學名 | Granite |
| 分類 | 火成岩（深成岩） |
| 顏色 | 灰、粉紅、白等顆粒狀斑駁色澤 |
| 紋理 | 粗粒結晶，礦物顆粒明顯可見 |
| 硬度 | 6～7（整體，因含多種礦物） |
| 密度 | 2.65～2.75 g/cm³ |
| 光澤 | 顆粒晶面反光 |
| 成分 | 石英、長石、雲母 |
| 形成方式 | 岩漿在地底深處緩慢冷卻結晶而成 |
| 地質年代 | 各地質年代 |
| 世界產地 | 中國、印度、巴西、北歐 |
| 台灣產地 | 台灣本島露頭少見，金門、馬祖為典型花崗岩地質 |
| 收藏價值 | 一般，多作建材用途 |
| 用途 | 建材、紀念碑、廚房檯面 |
| 趣味知識 | 金門是花崗岩地質，與台灣本島以變質岩、沉積岩為主明顯不同，是辨認「金門石」的特徵 |

## 4. 玄武岩 Basalt

| 欄位 | 內容 |
|---|---|
| 中文名稱 | 玄武岩 |
| 英文名稱 | Basalt |
| 學名 | Basalt |
| 分類 | 火成岩（噴出岩） |
| 顏色 | 黑至深灰 |
| 紋理 | 細粒緻密，常見氣孔構造 |
| 硬度 | 約 6 |
| 密度 | 2.9～3.0 g/cm³ |
| 光澤 | 暗淡至微光 |
| 成分 | 富含輝石、橄欖石、斜長石 |
| 形成方式 | 岩漿噴出地表快速冷卻凝固 |
| 地質年代 | 火山活動時期，各地質年代皆有 |
| 世界產地 | 冰島、夏威夷、印度德干高原 |
| 台灣產地 | 澎湖群島（著名柱狀玄武岩景觀）、基隆嶼 |
| 收藏價值 | 一般，具完整柱狀節理的標本具展示價值 |
| 用途 | 建材、道路鋪面、澎湖傳統石滬與石牆 |
| 趣味知識 | 玄武岩冷卻收縮時常形成六角形柱狀節理，澎湖與愛爾蘭「巨人堤道」都是世界知名景觀 |

## 5. 黑曜岩 Obsidian

| 欄位 | 內容 |
|---|---|
| 中文名稱 | 黑曜岩 |
| 英文名稱 | Obsidian |
| 學名 | Obsidian |
| 分類 | 火成岩（噴出岩，天然玻璃） |
| 顏色 | 黑色，偶見紅棕、條紋 |
| 紋理 | 均質無結晶構造，斷口平滑鋒利 |
| 硬度 | 5～5.5 |
| 密度 | 約 2.4 g/cm³ |
| 光澤 | 玻璃光澤 |
| 成分 | 富含 SiO₂ 的火山玻璃，非晶質 |
| 形成方式 | 高黏度熔岩快速冷卻，礦物來不及結晶 |
| 地質年代 | 多見於年輕火山地質年代 |
| 世界產地 | 美國黃石公園、冰島、墨西哥、日本 |
| 台灣產地 | 天然產地罕見，市面多為進口或人工玻璃仿製品 |
| 收藏價值 | 中等，斷口完整鋒利者具工藝與收藏價值 |
| 用途 | 史前石器工具、裝飾品、刀刃 |
| 趣味知識 | 斷口極為鋒利，史前人類曾用來製作切割工具，銳利度可媲美現代手術刀 |

## 6. 大理岩 Marble

| 欄位 | 內容 |
|---|---|
| 中文名稱 | 大理岩 |
| 英文名稱 | Marble |
| 學名 | Marble |
| 分類 | 變質岩 |
| 顏色 | 白、灰，常帶各色雜質紋理 |
| 紋理 | 細至中粒結晶，拋光後呈現柔和光澤 |
| 硬度 | 3～4 |
| 密度 | 2.6～2.85 g/cm³ |
| 光澤 | 拋光後呈亮麗光澤 |
| 成分 | 主要為方解石 CaCO₃，由石灰岩變質而成 |
| 形成方式 | 石灰岩經高溫高壓變質作用重結晶 |
| 地質年代 | 台灣大理岩主要形成於中新世 |
| 世界產地 | 義大利卡拉拉、希臘 |
| 台灣產地 | 花蓮太魯閣峽谷，為全球知名大理岩地形代表 |
| 收藏價值 | 一般，特殊紋理或稀有色澤標本較具價值 |
| 用途 | 建材、雕刻藝術、太魯閣石雕工藝 |
| 趣味知識 | 太魯閣峽谷的絕壁景觀，以及砂卡礑溪清澈的藍綠色溪水，都與大理岩地質密切相關 |

## 7. 蛇紋岩 Serpentinite

| 欄位 | 內容 |
|---|---|
| 中文名稱 | 蛇紋岩 |
| 英文名稱 | Serpentinite |
| 學名 | Serpentinite |
| 分類 | 變質岩 |
| 顏色 | 墨綠、暗綠帶斑紋，似蛇皮 |
| 紋理 | 蠟狀質地，常見滑感表面 |
| 硬度 | 2.5～4 |
| 密度 | 2.5～2.7 g/cm³ |
| 光澤 | 蠟狀至油脂光澤 |
| 成分 | 蛇紋石群礦物，由橄欖岩經水化變質而成 |
| 形成方式 | 地函橄欖岩經海水熱液蝕變作用生成 |
| 地質年代 | 與中央山脈東側地質帶相關 |
| 世界產地 | 美國加州、義大利、日本 |
| 台灣產地 | 花蓮豐田、卓溪一帶為主要產地 |
| 收藏價值 | 高——1990 年代票選為台灣「省石」，優質標本具收藏價值 |
| 用途 | 建材、玉石加工（豐田玉即與蛇紋岩相關）、觀賞石 |
| 趣味知識 | 因表面紋理酷似蛇皮而得名，是唯一被官方票選為台灣代表岩石的種類 |

## 8. 板岩 Slate

| 欄位 | 內容 |
|---|---|
| 中文名稱 | 板岩 |
| 英文名稱 | Slate |
| 學名 | Slate |
| 分類 | 變質岩 |
| 顏色 | 灰黑、深灰 |
| 紋理 | 具明顯劈理，可剝成薄片 |
| 硬度 | 3～4 |
| 密度 | 2.7～2.8 g/cm³ |
| 光澤 | 絹絲至暗淡光澤 |
| 成分 | 由頁岩（泥岩）經低度變質作用而成 |
| 形成方式 | 頁岩受區域變質作用，礦物定向排列產生劈理 |
| 地質年代 | 台灣板岩主要分布於中央山脈變質岩帶 |
| 世界產地 | 英國威爾斯、中國 |
| 台灣產地 | 太魯閣、能高越嶺一帶廣泛分布 |
| 收藏價值 | 一般，具明顯劈理或化石痕跡者較特別 |
| 用途 | 建材（石板屋屋頂）、傳統原住民石板屋建材 |
| 趣味知識 | 排灣族、魯凱族傳統石板屋大量使用當地板岩堆砌而成，是重要的文化資產 |

## 9. 礫岩 Conglomerate

| 欄位 | 內容 |
|---|---|
| 中文名稱 | 礫岩 |
| 英文名稱 | Conglomerate |
| 學名 | Conglomerate |
| 分類 | 沉積岩 |
| 顏色 | 混合色，依所含礫石種類而異 |
| 紋理 | 大小不一的圓形礫石膠結而成，表面粗糙 |
| 硬度 | 依膠結物與礫石成分而異，約 3～7 |
| 密度 | 約 2.3～2.7 g/cm³ |
| 光澤 | 無特定光澤 |
| 成分 | 磨圓礫石＋碳酸鈣或矽質膠結物 |
| 形成方式 | 河流、海濱等高能量環境搬運磨圓的碎屑沉積膠結而成 |
| 地質年代 | 台灣北海岸礫岩多形成於中新世 |
| 世界產地 | 全球河流沖積地形皆可見 |
| 台灣產地 | 野柳、南雅奇岩海岸地形 |
| 收藏價值 | 一般，內含特殊化石或礦物者較有價值 |
| 用途 | 建材、地質教育觀察 |
| 趣味知識 | 野柳知名的「女王頭」正是礫岩經海浪與風化長期侵蝕形成的蕈狀岩，且侵蝕仍在持續，頸部逐年變細 |

## 10. 北投石 Hokutolite

| 欄位 | 內容 |
|---|---|
| 中文名稱 | 北投石 |
| 英文名稱 | Hokutolite |
| 學名 | Hokutolite（硫酸鉛鋇礦系列） |
| 分類 | 礦物（稀有放射性礦物） |
| 顏色 | 灰白至黃褐色，常見同心圓層狀構造 |
| 紋理 | 層狀包覆於礫石表面 |
| 硬度 | 3～3.5 |
| 密度 | 約 4.4～4.7 g/cm³（含鉛、鋇，密度偏高） |
| 光澤 | 絲絹至珍珠光澤 |
| 成分 | 硫酸鉛鋇礦物，含微量放射性元素鐳 |
| 形成方式 | 溫泉水中的鉛、鋇離子與硫酸根結合，層層沉積包覆於溫泉水道礫石表面 |
| 地質年代 | 現代溫泉沉積作用，持續形成中 |
| 世界產地 | 全球僅台灣北投與日本秋田縣玉川溫泉兩處發現 |
| 台灣產地 | 台北市北投溫泉區（北投溪） |
| 收藏價值 | 極高——全球稀有、產地僅兩處，具放射性需妥善保存管理，是台灣礦物界珍寶 |
| 用途 | 礦物學研究、地質教育展示（北投溫泉博物館可見展示），因具放射性不作一般飾品配戴 |
| 趣味知識 | 1905 年由日本學者岡本要八郎在北投溪發現，是唯一以台灣地名命名的礦物 |

## 分類統計（對應 GDD 第 7.3 節圖鑑分類）

| 岩石/礦物種類 | 涵蓋條目 |
|---|---|
| 火成岩 | 花崗岩、玄武岩、黑曜岩 |
| 沉積岩 | 礫岩 |
| 變質岩 | 大理岩、蛇紋岩、板岩 |
| 礦物 | 石英、紫水晶、北投石 |

台灣在地產地涵蓋：花蓮（大理岩、蛇紋岩）、澎湖（玄武岩）、野柳（礫岩）、太魯閣／能高（板岩）、金門（花崗岩）、北投（北投石）——可直接對應 GDD 第 11 章「探索與召喚系統」的地區分布設計。
